# AI Chest X-Ray Disease Detection System - Web Application

This is a complete, working Flask-based medical web application that analyzes chest X-ray images to detect Normal, Pneumonia, or Tuberculosis pulmonary conditions. It wraps a trained Random Forest model backend classification pipeline in a premium, glassmorphic medical-themed frontend dashboard.

## ⚙️ Installation & Setup

Ensure you have Python 3.9+ installed on your system.

### 1. Set Up Virtual Environment (Optional but Recommended)
Initialize a virtual environment in the project directory:
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### 2. Install Required Dependencies
Install all package requirements declared in `requirements.txt`:
```bash
pip install -r requirements.txt
```

## 🚀 Running the Web Application

To launch the local development server:
```bash
python app.py
```

By default, Flask starts up on `http://127.0.0.1:5000/`.
1. Open this address in your web browser.
2. Click one of the three **Preset Demo Cases** to test classification outputs immediately.
3. Drag & drop or select your own custom `.jpg`, `.jpeg`, or `.png` X-ray scans.
4. Click **Analyze X-Ray** to see real-time status logging and animated metrics predicted by the model.

## 🩺 Pipeline Details

### Image Preprocessing
Before being fed to the Random Forest model, the uploaded image is:
1. Opened as a Pillow image.
2. Converted to grayscale (`"L"`).
3. Resized to `64 × 64` pixels.
4. Converted into a NumPy array and flattened into shape `(1, 4096)`.

### Inference & Safe Class Mapping
The backend calculates class probabilities using:
* `prediction = model.predict(x)[0]`
* `probabilities = model.predict_proba(x)[0]`

Outputs map dynamically using `model.classes_` to ensure that class indexes (0 = NORMAL, 1 = PNEUMONIA, 2 = TUBERCULOSIS) map safely to confidence scores, preventing runtime index crashes.
